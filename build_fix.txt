ERROR

Vercel TypeScript build failure in web/src/components/LandingAdminPreview.tsx line 51:
  "Property 'gradedPicks' does not exist on type ... MLBSubscriberView props."

ROOT CAUSE
----------
MLBSubscriberView.tsx was substantially rewritten in a prior session but never
committed. The rewrite replaced an async Supabase fetch pattern (useEffect +
getMLBSubscriberResults) with a gradedPicks?: MLBPickDetail[] prop + client-side
computeSegment logic. Three dependent changes were also left uncommitted.

Vercel built from HEAD, which still had the old MLBSubscriberView signature
(no gradedPicks prop). LandingAdminPreview.tsx passed gradedPicks={data.gradedPicks}
to it, causing the type error. The prop is genuinely needed — SubscriberTrackRecord
uses gradedPicks to compute segment stats and CLV beat-rate. Simply removing the
prop would have silenced the error but broken the track record panel.

DECISION
--------
Committed the correct local implementation. gradedPicks IS the right prop —
the subscriber view uses it to drive the full historical track record panel
(win/loss record, ROI, avg edge, avg CLV, CLV beat-rate by date range).
